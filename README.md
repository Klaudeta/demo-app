# Demo Application for Vaadin Flow

This project is a Vaadin Flow demo on basic features of grid, field type components, dynamic rendering of elements and notification/confirmation dialog.

It displays basic grid with multiple sorting enabled, column filtering and row selection.

It also makes use of the combobox component and its value change listener, dynamic rendering and readonly features of the components.

And as last thing the use of notification and confirmation dialogs triggered by user interaction


To access it directly from github, clone the repository and import the project to the IDE of your choice as a Maven project. You need to have Java 8 installed.

Run using `mvn jetty:run` and open [http://localhost:8080](http://localhost:8080) in browser.


Branching information:
* `master` the latest version of the starter, using latest platform snapshot
